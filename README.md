📌 Leads Tracker Web App

A simple and efficient web application to save, manage, and track useful leads (like URLs, notes, or resources). This project is perfect for improving productivity and practicing JavaScript fundamentals.

🚀 Features
✅ Add new leads (URLs or text)
📂 Save leads in browser storage
🔁 Persistent data using Local Storage
❌ Delete individual leads
🧹 Clear all leads
🔗 Clickable links open in new tab
⚡ Fast and lightweight UI
🛠️ Tech Stack
HTML – Structure
CSS – Styling
JavaScript – Functionality
Local Storage API – Data persistence

⚙️ Installation & Usage

Clone the repository:

git clone https://github.com/your-username/leads-tracker.git

Open the project folder:

cd leads-tracker
Open index.html in your browser
🧩 If Using as Chrome Extension
Open Google Chrome

Go to:

chrome://extensions/
Enable Developer Mode
Click Load unpacked
Select your project folder
📌 How It Works
Enter a lead (URL/text) in input field
Click Save Input → stored in Local Storage
Click Save Tab → saves current browser tab
Leads are displayed as clickable links
Data remains even after refreshing
